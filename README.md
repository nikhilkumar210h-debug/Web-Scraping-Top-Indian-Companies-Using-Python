# Web Scraping Top Indian Companies Using Python

## 📌 Project Overview

This project demonstrates how to perform web scraping using Python. Data is collected from AmbitionBox and transformed into a structured dataset using Pandas.

The project extracts information about thousands of companies, including ratings, reviews, salaries, interview experiences, job openings, benefits, and more.

The final dataset is cleaned and exported as a CSV file for further analysis and visualization.

---

## 🎯 Objectives

- Learn web scraping with Python
- Extract data from real websites
- Parse HTML using BeautifulSoup
- Collect data from multiple pages
- Build structured datasets
- Perform basic data cleaning

---

## 🔍 Workflow

1. Send HTTP requests to company listing pages
2. Parse HTML content using BeautifulSoup
3. Extract company information
4. Loop through multiple pages
5. Store data in Python dictionaries
6. Convert data into a Pandas DataFrame
7. Clean numerical values
8. Export the dataset to CSV format
